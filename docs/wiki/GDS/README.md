# Wiki de GDS TravelHub 🚀

Bienvenido a la Base de Conocimiento Técnica de GDS de TravelHub. Aquí consolidamos las "mañas", reglas de negocio y especificaciones técnicas extraídas de manuales oficiales y experiencia operativa.

## Estructura de la Wiki

- [**Amadeus**](Amadeus/README.md): Reglas de emisión, reemisión y lectura de reportes.
- [**Sabre**](Sabre/README.md): Comandos críticos y estructura de itinerarios complejos.
- [**KIU**](KIU/README.md): Particularidades del sistema regional y sufijos de reemisión.

## Motor de Reconciliación

Toda la inteligencia documentada aquí alimenta directamente al **Motor de Reconciliación IA**, permitiendo que el sistema entienda discrepancias de céntimos basadas en reglas específicas de cada proveedor.
